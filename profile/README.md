ชมรม KU Tech ประกาศรับสมัครสมาชิกรุ่นแรก เพื่อขอจัดตั้งกลุ่มกิจกรรม (ชมรม) กับอบก. โดยไม่จำกัดจำนวน คณะ, สาขา เพราะเราชื่อว่าทุกคนพัฒนาตัวเองได้ตลอดเวลา !

## ⚡️วัตถุประสงค์

เพื่อรวบรวมนิสิตจากทุกคณะสาขาที่สนใจด้านเทคโนโลยี มาระดมความคิดร่วมกันแก้ปัญหา (Hackthon) ต่าง ๆ โดยใช้องค์ความรู้จากสาขาที่ตนเองถนัดเช่น ด้านการบริการธุรกิจ, ด้านวิศวะกรรม แล้วนำมาสร้าง Product ให้เกิดขึ้นและใช้งานได้จริง พร้อมเผยแพร่ความรู้ด้านเทคโนโลยีสู่สังคม

## ☀️ ตัวอย่างกิจกรรม

- KU Hackathon : ร่วมกันระดมความคิดแก้ไขปัญหาต่าง ๆ ภายในมหาวิทยาลัยด้วยเทคโนโลยี
- KU Tech Camp : ออกค่ายโรงเรียนมัธยมเพื่อสอนน้องด้านโค้ดเบื้องต้น
